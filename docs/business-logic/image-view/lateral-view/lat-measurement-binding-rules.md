# 侧位 X 光片测量项绑定规则

## 1. 适用范围

本文档用于定义侧位 X 光片中 AI 检测点、手工基础标注、测量项、工具面板入口、更新规则和删除规则之间的绑定关系。

## 2. 基础标注对象

### 2.1 AI 检测对象

侧位片 AI 检测提供以下对象：

| 对象 | 点位 | 说明 |
|---|---:|---|
| C7 | 4 点 | C7 椎体四角点 |
| T1 到 T12 | 每节 4 点 | 胸椎椎体四角点 |
| L1 到 L5 | 每节 4 点 | 腰椎椎体四角点 |
| S1 | 2 点 | S1 上终板两点 |
| CFH | 1 点 | 股骨头中心单点坐标 |
| 标准距离 | 系统提供 | 距离类指标使用标准距离换算 |

### 2.2 手工基础标注对象

| 对象 | 点位 | 说明 |
|---|---:|---|
| C2 | 4 点 | C2 椎体四角点由用户手工标注 |
| 辅助图形 | 不固定 | 用户自由绘制 |

C2 不由未来 AI 检测提供。C2-C7 CL 只需要 C2 的目标终板两点参与计算，但系统仍按 C2 四角点进行手工标注，便于统一椎体数据结构和后续扩展。

### 2.3 CFH 规则

CFH 是单点坐标，直接作为股骨头中心参与计算。系统禁止把 CFH 按两侧股骨头点的中点重新计算。

依赖 CFH 的测量项包括：

| 测量项 | CFH 用途 |
|---|---|
| TPA | 构成 T1、S1、CFH 之间的角度关系 |
| PI | 构成 S1 上终板中点到 CFH 的连线 |
| PT | 构成 S1 上终板中点到 CFH 的连线 |

### 2.4 S1 规则

S1 使用 AI 检测提供的 S1 上终板两点。
依赖 S1 上终板两点的测量项包括：

| 测量项 | S1 用途 |
|---|---|
| LL L1-S1 | S1 上终板线 |
| LL L4-S1 | S1 上终板线 |
| TPA | S1 上终板中点 |
| SVA | S1 上终板后端点 |
| PI | S1 上终板线和中点 |
| PT | S1 上终板中点 |
| SS | S1 上终板线 |

S1 上终板两点需要具备可区分的前端点和后端点语义，或由系统根据影像方向确定前后关系。SVA 依赖 S1 上终板后端点。

### 2.5 标准距离规则

SVA 是距离类指标，输出时使用系统标准距离进行换算。

若后续存在其他距离类测量项，也统一使用标准距离输出实际长度。

## 3. 测量项唯一性

### 3.1 全局唯一测量项

以下测量项在一张侧位片中最多出现一次：

| 测量项 | 唯一性 |
|---|---|
| T1 Slope | 全局唯一 |
| C2-C7 CL | 全局唯一 |
| TK T2-T5 | 全局唯一 |
| TK T5-T12 | 全局唯一 |
| T10-L2 | 全局唯一 |
| LL L1-S1 | 全局唯一 |
| LL L1-L4 | 全局唯一 |
| LL L4-S1 | 全局唯一 |
| TPA | 全局唯一 |
| SVA | 全局唯一 |
| PI | 全局唯一 |
| PT | 全局唯一 |
| SS | 全局唯一 |

### 3.2 非全局唯一测量项

| 测量项 | 唯一性 |
|---|---|
| 椎体中心 | 全局可多个，建议每个椎体最多一个 |
| 辅助图形 | 可多个 |

## 4. 五类绑定规则总览

| 类别 | 规则名称 | 点位来源 | 用户参与 | 唯一性 | 当前适用项 |
|---|---|---|---|---|---|
| A | 全自动全局唯一测量项 | 全部来自 AI 检测点 | 不需要 | 全局唯一 | T1 Slope、TK T2-T5、TK T5-T12、T10-L2、LL L1-S1、LL L1-L4、LL L4-S1、TPA、SVA、PI、PT、SS |
| B | 选择式 AI 绑定测量项 | 全部来自 AI 检测点 | 用户选择绑定对象 | 非全局唯一 | 椎体中心 |
| C | 部分 AI 加部分手工，全局唯一测量项 | AI 检测点加手工点 | 用户补充手工基础标注 | 全局唯一 | C2-C7 CL |
| D | 部分 AI 加部分手工，全局不唯一测量项 | AI 检测点加手工点 | 用户补充手工基础标注 | 非全局唯一 | 当前暂无 |
| E | 纯手工测量项或辅助图形 | 全部来自用户手工点 | 用户自由绘制 | 按对象配置 | 辅助图形；C2 四角点作为手工基础标注对象处理 |

## 5. A 类：全自动全局唯一测量项

### 5.1 条件

A 类测量项满足以下条件：

| 条件 | 说明 |
|---|---|
| 点位来源 | 全部依赖 AI 检测点 |
| 用户操作 | 用户不需要手工绘制测量项 |
| 唯一性 | 一张影像中最多一个 |
| 工具面板 | 不开放手工绘制入口 |
| 创建规则 | 依赖点全部存在时，系统自动生成测量项 |
| 更新规则 | 依赖点变化时，测量项同步更新 |
| 删除规则 | 依赖点删除时，测量项删除或置为失效，并提示用户 |

### 5.2 当前适用项

| 测量项 | 依赖点 | 说明 |
|---|---|---|
| T1 Slope | T1 上终板两点 | 由 T1 四角点取得上终板线 |
| TK T2-T5 | T2 上终板两点，T5 下终板两点 | 由 T2 和 T5 四角点取得端板线 |
| TK T5-T12 | T5 上终板两点，T12 下终板两点 | 由 T5 和 T12 四角点取得端板线 |
| T10-L2 | T10 上终板两点，L2 下终板两点 | 由 T10 和 L2 四角点取得端板线 |
| LL L1-S1 | L1 上终板两点，S1 上终板两点 | L1 来自四角点，S1 来自 AI 的 S1 上终板两点 |
| LL L1-L4 | L1 上终板两点，L4 下终板两点 | 由 L1 和 L4 四角点取得端板线 |
| LL L4-S1 | L4 上终板两点，S1 上终板两点 | L4 来自四角点，S1 来自 AI 的 S1 上终板两点 |
| TPA | T1 中心，S1 上终板中点，CFH | T1 中心由 T1 四角点确定，CFH 为单点坐标 |
| SVA | C7 中心，S1 上终板后端点，标准距离 | C7 中心由 C7 四角点确定，距离输出使用标准距离 |
| PI | S1 上终板线，S1 上终板中点，CFH | CFH 为单点坐标 |
| PT | S1 上终板中点，CFH，垂直方向 | CFH 为单点坐标 |
| SS | S1 上终板线，水平参考方向 | S1 上终板两点由 AI 提供 |

### 5.3 工具面板规则

A 类测量项不出现在手工绘制工具面板中。

若产品需要允许用户控制显示隐藏，应使用显示开关，不使用创建入口。

### 5.4 删除和失效规则

当某个 AI 点被删除时：

| 情况 | 处理 |
|---|---|
| 依赖该点的 A 类测量项存在 | 删除或置为失效 |
| 用户撤销删除 | 恢复依赖关系和测量结果 |
| 无法恢复依赖点 | 保持失效状态 |

提示示例：

| 场景 | Toast 文案 |
|---|---|
| T1 被删除 | 因 T1 检测点缺失，T1 Slope 已失效 |
| S1 上终板点缺失 | 因 S1 上终板点缺失，相关骨盆参数已失效 |
| CFH 缺失 | 因 CFH 缺失，TPA、PI、PT 已失效 |

## 6. B 类：选择式 AI 绑定测量项

### 6.1 条件

B 类测量项满足以下条件：

| 条件 | 说明 |
|---|---|
| 点位来源 | 全部来自 AI 检测点 |
| 用户操作 | 用户选择需要绑定的 AI 对象 |
| 唯一性 | 一张影像中可存在多个实例 |
| 工具面板 | 开放选择式入口 |
| 创建规则 | 用户选择对象后生成测量项 |
| 更新规则 | 被绑定 AI 对象变化时，该测量项同步更新 |
| 删除规则 | 被绑定 AI 对象删除时，该实例删除或置为失效 |

### 6.2 当前适用项

| 测量项 | 依赖对象 | 唯一性建议 | 说明 |
|---|---|---|---|
| 椎体中心 | 用户选择的 C7、T1 到 T12、L1 到 L5 中任一椎体四角点 | 每个椎体最多一个 | 系统根据椎体四角点计算中心 |

### 6.3 工具面板规则

椎体中心工具满足以下条件时开放：

| 条件 | 处理 |
|---|---|
| 存在至少一个可用椎体 | 开放椎体中心工具 |
| 某个椎体已存在椎体中心 | 该椎体不可再次创建椎体中心 |
| 某个椎体尚未创建椎体中心 | 该椎体可创建椎体中心 |

### 6.4 删除和失效规则

| 场景 | 处理 |
|---|---|
| 被绑定椎体点位调整 | 椎体中心同步更新 |
| 被绑定椎体删除 | 对应椎体中心删除或置为失效 |
| 其他椎体删除 | 不影响该椎体中心 |

## 7. C 类：部分 AI 加部分手工，全局唯一测量项

### 7.1 条件

C 类测量项满足以下条件：

| 条件 | 说明 |
|---|---|
| 点位来源 | 一部分来自 AI 检测点，一部分来自手工基础标注 |
| 用户操作 | 用户补充必要的手工基础标注 |
| 唯一性 | 一张影像中最多一个 |
| 工具面板 | 依赖 AI 点存在，且该测量项未存在时开放 |
| 创建规则 | 用户完成手工基础标注后生成测量项 |
| 更新规则 | AI 点或手工点变化时，测量项同步更新 |
| 删除规则 | AI 点或手工点缺失时，测量项删除或置为失效 |

### 7.2 当前适用项

| 测量项 | AI 依赖 | 手工依赖 | 说明 |
|---|---|---|---|
| C2-C7 CL | C7 四角点 | C2 四角点 | C2-C7 CL 使用 C2 的目标终板两点和 C7 的目标终板两点 |

### 7.3 C2 手工标注规则

| 规则 | 说明 |
|---|---|
| C2 标注对象 | C2 四角点 |
| 创建方式 | 用户手工标注 |
| 唯一性 | 一张影像中最多一个 C2 手工标注对象 |
| 与 C2-C7 CL 的关系 | C2-C7 CL 依赖 C2 手工标注对象 |
| 点位更新 | 用户调整 C2 点位后，C2-C7 CL 同步更新 |
| 点位删除 | C2 手工标注对象删除后，C2-C7 CL 删除或置为失效 |

### 7.4 C2-C7 CL 工具面板规则

| 条件 | C2 手工标注入口 | C2-C7 CL 测量入口 |
|---|---|---|
| C7 存在，C2 不存在，C2-C7 CL 不存在 | 开放 | 不开放或提示先标 C2 |
| C7 存在，C2 存在，C2-C7 CL 不存在 | 不开放 | 开放 |
| C7 存在，C2 存在，C2-C7 CL 已存在 | 不开放 | 不开放 |
| C7 缺失 | 可按产品策略开放 C2 标注 | 不开放 |

### 7.5 删除和失效规则

| 场景 | 处理 |
|---|---|
| C7 被删除 | C2-C7 CL 删除或置为失效 |
| C2 手工标注对象被删除 | C2-C7 CL 删除或置为失效 |
| C7 点位调整 | C2-C7 CL 同步更新 |
| C2 点位调整 | C2-C7 CL 同步更新 |

提示示例：

| 场景 | Toast 文案 |
|---|---|
| C7 缺失 | 因 C7 检测点缺失，C2-C7 CL 已失效 |
| C2 缺失 | 因 C2 手工标注缺失，C2-C7 CL 已失效 |

## 8. D 类：部分 AI 加部分手工，全局不唯一测量项

### 8.1 条件

D 类测量项满足以下条件：

| 条件 | 说明 |
|---|---|
| 点位来源 | 一部分来自 AI 检测点，一部分来自手工点 |
| 用户操作 | 用户补充必要手工点 |
| 唯一性 | 一张影像中允许多个实例 |
| 工具面板 | 依赖 AI 点存在时开放 |
| 更新规则 | AI 点或手工点变化时，仅对应实例同步更新 |
| 删除规则 | AI 点或手工点缺失时，仅对应实例删除或置为失效 |

### 8.2 当前适用项

当前侧位固定测量项中暂无 D 类测量项。

若后续新增某个测量项同时满足以下条件，则归入 D 类：

| 条件 |
|---|
| 依赖 AI 检测点 |
| 需要用户补充手工点 |
| 一张影像中允许创建多个实例 |

## 9. E 类：纯手工测量项或辅助图形

### 9.1 条件

E 类对象满足以下条件：

| 条件 | 说明 |
|---|---|
| 点位来源 | 不依赖 AI 检测点 |
| 用户操作 | 用户手工绘制 |
| 唯一性 | 按对象类型配置 |
| 工具面板 | 按唯一性和业务条件开放 |
| 更新规则 | 用户调整手工点后更新 |
| 删除规则 | 用户删除对象后，依赖该对象的测量项删除或置为失效 |

### 9.2 当前适用项

| 对象 | 说明 |
|---|---|
| 辅助图形 | 用户自由绘制，可多次添加 |
| C2 四角点 | 作为手工基础标注对象处理，服务于 C2-C7 CL |

当前侧位固定测量项列表中暂无纯手工全局唯一测量项。

## 13. 删除和失效规则汇总

### 13.1 AI 点删除

| 被删除对象 | 处理 |
|---|---|
| C7 | SVA 和 C2-C7 CL 删除或失效；C7 椎体中心删除或失效 |
| T1 | T1 Slope 和 TPA 删除或失效；T1 椎体中心删除或失效 |
| T2 | TK T2-T5 删除或失效；T2 椎体中心删除或失效 |
| T5 | TK T2-T5 和 TK T5-T12 删除或失效；T5 椎体中心删除或失效 |
| T10 | T10-L2 删除或失效；T10 椎体中心删除或失效 |
| T12 | TK T5-T12 删除或失效；T12 椎体中心删除或失效 |
| L1 | LL L1-S1 和 LL L1-L4 删除或失效；L1 椎体中心删除或失效 |
| L2 | T10-L2 删除或失效；L2 椎体中心删除或失效 |
| L4 | LL L1-L4 和 LL L4-S1 删除或失效；L4 椎体中心删除或失效 |
| S1 上终板点 | LL L1-S1、LL L4-S1、TPA、SVA、PI、PT、SS 删除或失效 |
| CFH | TPA、PI、PT 删除或失效 |

### 13.2 手工对象删除

| 被删除对象 | 处理 |
|---|---|
| C2 四角点 | C2-C7 CL 删除或失效 |
| 辅助图形 | 显式依赖该辅助图形的对象删除或失效 |

## 15. 实施方案

1. 将所有自动测量项和手工测量项统一放入依赖关系图。
2. A 类测量项只在工具菜单中留点击创建入口，并提供可用状态显示，用于从列表项删除后恢复。
3. C2 作为手工基础标注对象单独管理，C2-C7 CL 依赖该对象。
4. CFH 只作为单点坐标使用，相关测量项不得自动改用双侧股骨头中点。
5. SVA 使用系统标准距离输出实际长度。
6. 所有依赖点调整时，相关测量项同步更新。
7. 所有依赖点删除时，相关测量项删除或置为失效，并通过 Toast 提醒用户。
