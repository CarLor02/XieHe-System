# 正位测量项绑定规则文档

## 1. 文档范围

本文档用于正位 X 光片中的测量项绑定规则，包括 AI 检测点、测量项唯一性、工具面板开放条件、依赖更新规则和删除规则。
当前系统中，一张影像包含多个椎体对象，例如 T1 到 T12、L1 到 L4。每个椎体由 AI 自动检测四个角点。除此之外，AI 还支持若干骨盆、锁骨和 C7 相关检测点。

## 2. 当前 AI 检测能力

| AI 检测对象 | 检测内容 | 相关测量项 |
|---|---|---|
| T1 到 T12 四角点 | 每个胸椎 4 个角点 | T1 Tilt、Cobb、椎体中心、TTS、AVT |
| L1 到 L4 四角点 | 每个腰椎 4 个角点 | Cobb、椎体中心、TTS、AVT |
| C7 四角点 | C7 椎体 4 个角点 | TS |
| IL、IR | 左右骨盆参考点 | PO |
| SL、SR | 左右骶骨参考点 | CSS、TTS、AVT |
| CL、CR | 左右锁骨参考点 | CA |

### 2.1 检测点标注功能
允许用户手动添加关键点，可添加的关键点目前定义为AI检测中的所有关键点，存在的关键点不能重复添加, 允许添加缺失的关键点, 所有存在的关键点做成状态机
添加关键点的菜单与测量标注菜单放在一起，做成可切换的Tab
对于椎体检测点，应该做成大类，比如 T1, 用户点击时，弹出一个小的列表overlay, 确定是四角点里具体哪个点

## 3. 当前测量项唯一性

### 3.1 全局唯一测量项

以下测量项在一张影像中只能出现一次。若已存在，工具面板中不再允许再次添加。

| 测量项 | 唯一性 |
|---|---|
| T1 Tilt | 全局唯一 |
| CA | 全局唯一 |
| PO | 全局唯一 |
| CSS | 全局唯一 |
| AVT | 全局唯一 |
| TTS | 全局唯一 |
| LLD | 全局唯一 |
| TS | 全局唯一 |

### 3.2 非全局唯一测量项

以下测量项或图形在一张影像中可以出现多次，但可以设置局部唯一限制。

| 测量项或图形 | 唯一性建议 |
|---|---|
| Cobb | 可多次添加，建议按绑定组合限制重复 |
| 椎体中心 | 可多次添加，每个椎体最多一个 |
| 辅助图形 | 可多次添加 |

## 4. 五类绑定规则总览

| 类别 | 规则名称 | 核心条件 | 当前测量项 |
|---|---|---|---|
| A 类 | 全自动全局唯一测量项 | 依赖点全部由 AI 唯一确定，用户不需要选择对象或绘制点 | T1 Tilt、CA、PO、CSS、TS |
| B 类 | 选择式 AI 绑定，非全局唯一测量项 | 坐标全部来自 AI 点，但需要用户选择绑定对象，且允许多个实例 | Cobb、椎体中心 |
| C 类 | 选择式 AI 绑定，全局唯一测量项 | 坐标全部来自 AI 点，但需要用户选择部分绑定对象，且全局只能存在一个 | AVT、TTS |
| D 类 | AI 点加手工点混合测量项 | 一部分依赖 AI 点，一部分依赖用户手工点或手工图形 | 当前暂无确认项 |
| E 类 | 纯手工测量项或辅助图形 | 不依赖 AI 检测点，由用户手工绘制或确定 | LLD、辅助图形 |

## 5. A 类规则：全自动全局唯一测量项

### 5.1 规则条件

A 类测量项满足以下条件：

| 条件 | 说明 |
|---|---|
| 点位来源 | 全部由 AI 检测点确定 |
| 用户选择 | 不需要用户选择绑定对象 |
| 用户绘制 | 不允许用户手动绘制 |
| 唯一性 | 一张影像中最多一个 |
| 创建方式 | AI 依赖点存在后，系统自动生成或自动展示 |
| 更新方式 | AI 依赖点变化后，测量项自动更新 |
| 删除方式 | AI 依赖点删除后，测量项删除或置为失效 |
| 工具面板 | 不开放手工绘制入口 |

### 5.2 适用测量项

| 测量项     | AI 依赖          | 规则说明                  |
|---------|----------------|-----------------------|
| T1 Tilt | T1 四角点         | T1 四角点存在时自动确定         |
| CA      | CL、CR          | CL 和 CR 存在时自动确定       |
| PO      | IL、IR          | IL 和 IR 存在时自动确定       |
| CSS     | SL、SR          | SL 和 SR 存在时自动确定       |
| TS      | C7 四角点, SL, SR | C7 四角点, SL, SR存在时自动确定 |
| Cobb    | 两组四角点 | AI自动识别大于10度的两组终板并显示 | 

### 6.3 Cobb 规则说明

Cobb 允许在同一张影像中存在多个实例。不需要用户选择，系统根据被选椎体的 AI 四角点生成端板线并计算角度。
Cobb 的坐标来源全部来自 AI 检测点，自动识别大于10度的两椎体上下终板夹角并显示,当前的实现已经使用的是这种方式。
禁止完全重复的 Cobb 组合。重复判断建议包括：

| 重复判断字段 |
|---|
| 上端椎 |
| 上端椎端板方向 |
| 下端椎 |
| 下端椎端板方向 |

如果以上字段完全一致，则视为同一个 Cobb，不允许重复添加。

Cobb在结果列表里应显示使用的是哪两个椎体的上下终板，比如 `Cobb(T1-T12)`


### 5.3 工具面板规则

A 类测量项不在手工工具面板中提供自由绘制入口。系统根据 AI 检测结果自动创建或自动显示测量结果。
工具菜单可以显示 A 类测量项的状态入口；当用户删除某个自动测量项后，入口可用于从当前 AI 检测点恢复该测量项，但仍不允许用户手工绘制测量点。

### 5.4 更新规则

当 A 类测量项依赖的 AI 点发生调整时，对应测量项自动更新。用户不需要重新创建该测量项。

示例：

| 变化对象 | 影响测量项 |
|---|---|
| T1 四角点变化 | T1 Tilt 更新 |
| CL 或 CR 变化 | CA 更新 |
| IL 或 IR 变化 | PO 更新 |
| SL 或 SR 变化 | CSS 更新 |
| C7 四角点变化 | TS 更新 |

### 5.5 删除规则

当 A 类测量项依赖的 AI 点被删除时，对应测量项删除或置为失效，并向用户提示原因。

示例提示：

| 场景 | 提示文案 |
|---|---|
| CL 或 CR 缺失 | 因 CL 或 CR 检测点缺失，CA 已移除 |
| IL 或 IR 缺失 | 因 IL 或 IR 检测点缺失，PO 已移除 |
| SL 或 SR 缺失 | 因 SL 或 SR 检测点缺失，CSS 已移除 |
| C7 四角点缺失 | 因 C7 检测点缺失，TS 已移除 |

## 6. B 类规则：选择式 AI 绑定，非全局唯一测量项

### 6.1 规则条件

B 类测量项满足以下条件：

| 条件 | 说明 |
|---|---|
| 点位来源 | 全部由 AI 检测点推导 |
| 用户选择 | 需要用户选择绑定对象 |
| 用户绘制 | 不允许用户自由绘制测量点 |
| 唯一性 | 全局允许多个实例 |
| 创建方式 | 用户选择 AI 对象后生成 |
| 更新方式 | 被绑定 AI 点变化后，对应测量项自动更新 |
| 删除方式 | 被绑定 AI 点删除后，对应测量项删除或置为失效 |
| 工具面板 | 开放选择式入口 |

### 6.2 适用测量项

| 测量项 | 用户选择内容 | AI 依赖 | 唯一性建议 |
|---|---|---|---|
| 椎体中心 | 一个椎体 | 被选椎体四角点 | 每个椎体最多一个 |

### 6.4 椎体中心规则说明

椎体中心允许在一张影像中添加多个实例，但建议每个椎体最多创建一个椎体中心。
用户选择某个椎体后，系统根据该椎体四角点推导中心点。
示例：

| 场景 | 规则 |
|---|---|
| T7 椎体中心已存在 | 不允许再次为 T7 添加椎体中心 |
| T8 椎体中心不存在 | 允许为 T8 添加椎体中心 |
| T7 四角点调整 | T7 椎体中心自动更新 |
| T7 被删除 | T7 椎体中心删除或置为失效 |

## 7. C 类规则：选择式 AI 绑定，全局唯一测量项

### 7.1 规则条件

C 类测量项满足以下条件：

| 条件 | 说明 |
|---|---|
| 点位来源 | 全部由 AI 检测点或 AI 派生点确定 |
| 用户选择 | 需要用户选择部分绑定对象 |
| 用户绘制 | 不需要用户自由绘制测量点 |
| 唯一性 | 一张影像中最多一个 |
| 创建方式 | AI 基础依赖存在，且测量项尚不存在时，允许用户选择绑定对象创建 |
| 更新方式 | 被绑定 AI 点变化后，测量项自动更新 |
| 删除方式 | 被绑定 AI 点删除后，测量项删除或置为失效 |
| 工具面板 | 满足依赖且全局不存在时开放 |

### 7.2 适用测量项

| 测量项 | AI 基础依赖 | 用户选择内容 | 规则说明 |
|---|---|---|---|
| AVT | SL、SR，以及可用椎体四角点 | 一个顶椎中心 | SL 和 SR 确定 midline，用户选择顶椎中心后生成 AVT |
| TTS | SL、SR，以及可用椎体四角点 | 两个椎体中心 | SL 和 SR 确定骶骨垂线，用户选择两个椎体中心确定胸廓中线 |

### 7.3 TTS 规则说明

TTS 是全局唯一测量项。一张影像中最多存在一个 TTS。
TTS 的依赖关系分为两部分：

| 组成部分 | 来源 | 说明 |
|---|---|---|
| 骶骨垂线 | SL、SR | AI 根据 SL 和 SR 直接确定并绘制 |
| 胸廓中线 | 用户选择的两个椎体中心 | 椎体中心由对应椎体四角点推导 |

用户创建 TTS 时，只需要选择用于确定胸廓中线的两个椎体中心。系统不允许用户自由绘制胸廓中线的端点。
TTS 的端点不允许直接拖动；若需要调整 TTS，只能调整 SL、SR 或被选椎体的四角点，系统再自动更新 TTS。

### 7.4 TTS 工具面板规则

TTS 工具入口开放需同时满足以下条件：

| 条件 |
|---|
| 当前影像中不存在 TTS |
| SL 和 SR 均存在 |
| 至少存在两个可用于生成椎体中心的椎体 |

若以上条件不满足，TTS 工具入口隐藏或禁用。

### 7.5 TTS 更新规则

| 变化对象 | 更新结果 |
|---|---|
| SL 或 SR 调整 | 骶骨垂线更新，TTS 更新 |
| 被选椎体四角点调整 | 对应椎体中心更新，胸廓中线更新，TTS 更新 |
| 其他未绑定椎体调整 | 不影响当前 TTS |

### 7.6 TTS 删除规则

| 删除对象 | 处理方式 |
|---|---|
| SL 或 SR 被删除 | TTS 删除或置为失效 |
| 被选的任一椎体被删除 | TTS 删除或置为失效 |
| 未被 TTS 绑定的椎体被删除 | 不影响当前 TTS |

示例提示：

| 场景 | 提示文案 |
|---|---|
| SL 或 SR 缺失 | 因 SL 或 SR 检测点缺失，TTS 已移除 |
| 被选椎体缺失 | 因 TTS 绑定的椎体检测点缺失，TTS 已移除 |

### 7.7 AVT 规则说明

AVT 是半自动的全局唯一测量项。一张影像中最多存在一个 AVT。
AVT 的基础依赖为 SL、SR，以及用户选择的顶椎四角点。

| 组成部分 | 来源 | 说明 |
|---|---|---|
| midline | SL、SR | 取 SL、SR 连线中点，并经过该中点作中垂线 |
| 顶椎垂线 | 用户选择的顶椎中心 | 顶椎中心由对应椎体四角点推导，经过该中心作垂线 |
| AVT 距离 | midline 与顶椎垂线 | 计算两条垂线之间的水平距离 |

用户创建 AVT 时，只需要选择一个顶椎。系统不允许用户自由绘制顶椎中心或 midline 端点。

AVT 工具入口开放需同时满足以下条件：

| 条件 |
|---|
| 当前影像中不存在 AVT |
| SL 和 SR 均存在 |
| 至少存在一个可用于生成椎体中心的椎体 |

AVT 更新规则：

| 变化对象 | 更新结果 |
|---|---|
| SL 或 SR 调整 | midline 更新，AVT 更新 |
| 被选顶椎四角点调整 | 顶椎中心更新，AVT 更新 |
| 其他未绑定椎体调整 | 不影响当前 AVT |

AVT 删除规则：

| 删除对象 | 处理方式 |
|---|---|
| SL 或 SR 被删除 | AVT 删除或置为失效 |
| 被选顶椎被删除 | AVT 删除或置为失效 |
| 未被 AVT 绑定的椎体被删除 | 不影响当前 AVT |

## 8. D 类规则：AI 点加手工点混合测量项

### 8.1 规则条件

D 类测量项满足以下条件：

| 条件 | 说明 |
|---|---|
| 点位来源 | 一部分来自 AI 检测点，一部分来自用户手工点或手工图形 |
| 用户选择 | 可能需要用户选择 AI 对象 |
| 用户绘制 | 需要用户补充手工点、线或辅助图形 |
| 唯一性 | 按具体测量项配置，可以全局唯一，也可以多实例 |
| 创建方式 | AI 依赖满足后，用户补充手工对象创建 |
| 更新方式 | AI 点或手工对象变化后，测量项自动更新 |
| 删除方式 | 任一必要依赖删除后，测量项删除或置为失效 |
| 工具面板 | 根据 AI 依赖和唯一性判断是否开放 |

### 8.2 当前适用测量项

当前已明确的测量项中，暂无必须归入 D 类的项。

| 测量项 | 是否归入 D 类 | 原因 |
|---|---|---|
| AVT | 否 | AVT 的 midline 和顶椎垂线均由 AI 点或 AI 派生点确定，用户只选择顶椎中心 |
| TTS | 否 | TTS 的骶骨垂线和胸廓中线均由 AI 点或 AI 派生点确定，用户只选择椎体中心 |
| 椎体中心 | 否 | 椎体中心全部由被选椎体四角点推导 |
| LLD | 否 | 当前没有相关 AI 下肢点，按纯手工测量处理 |

### 8.3 预留规则

若后续新增测量项满足以下条件，则归入 D 类：

| 条件 |
|---|
| 依赖某些 AI 检测点 |
| 仍需要用户手工绘制额外点、线或图形 |
| 测量结果由 AI 依赖和手工依赖共同决定 |

示例场景：

| 场景 | 处理方式 |
|---|---|
| 某测量项依赖 C7 中心，同时需要用户手工标注躯干边界点 | 归入 D 类 |
| 某测量项依赖 SL、SR，同时需要用户手工绘制额外参考线 | 归入 D 类 |

## 9. E 类规则：纯手工测量项或辅助图形

### 9.1 规则条件

E 类对象满足以下条件：

| 条件 | 说明 |
|---|---|
| 点位来源 | 不依赖 AI 检测点 |
| 用户选择 | 用户自行确定点、线或图形 |
| 用户绘制 | 允许用户手工绘制 |
| 唯一性 | 按具体测量项配置 |
| 创建方式 | 用户从工具面板创建 |
| 更新方式 | 用户调整手工对象后更新 |
| 删除方式 | 手工对象删除后，对应测量项删除或置为失效 |
| 工具面板 | 按唯一性判断是否开放 |

### 9.2 适用测量项和图形

| 测量项或图形 | 唯一性 | 规则说明 |
|---|---|---|
| LLD | 全局唯一 | 当前无股骨头、踝关节或下肢端点 AI 检测，按用户手工测量处理 |
| 辅助线 | 可多次添加 | 用户自由绘制 |
| 辅助圆 | 可多次添加 | 用户自由绘制 |
| 辅助点 | 可多次添加 | 用户自由绘制 |
| 其他辅助图形 | 可多次添加 | 用户自由绘制 |

### 9.3 LLD 规则说明

LLD 当前按纯手工全局唯一测量项处理。一张影像中最多存在一个 LLD。

LLD 工具入口开放需满足：

| 条件 |
|---|
| 当前影像中不存在 LLD |

LLD 相关点、线或辅助图形由用户手工确定。若用户删除 LLD 依赖的手工对象，LLD 删除或置为失效。

### 9.4 辅助图形规则说明

辅助图形不参与全局唯一限制，默认允许多次添加。
辅助图形本身不依赖 AI 检测点。若某个测量项引用了辅助图形，则辅助图形成为该测量项的手工依赖。

## 10. 工具面板开放规则汇总

| 类别 | 开放规则 |
|---|---|
| A 类 | 不开放手工入口，系统自动生成或展示 |
| B 类 | 存在可选 AI 对象时开放选择式入口 |
| C 类 | AI 基础依赖存在，测量项全局不存在，且存在足够的可选 AI 对象时开放 |
| D 类 | AI 依赖存在，且满足该测量项唯一性限制时开放 |
| E 类 | 按唯一性限制开放，辅助图形默认始终开放 |

## 11. 删除和失效规则汇总

### 11.1 AI 检测点删除

| 类别 | 处理规则 |
|---|---|
| A 类 | 删除或失效依赖该 AI 点的全自动测量项 |
| B 类 | 只删除或失效绑定该 AI 点的实例 |
| C 类 | 删除或失效该全局唯一测量项 |
| D 类 | 删除或失效依赖该 AI 点的实例 |
| E 类 | 不受 AI 点删除影响 |

### 11.2 手工对象删除

| 对象 | 处理规则 |
|---|---|
| LLD 依赖的手工对象 | LLD 删除或置为失效 |
| 被其他测量项引用的辅助图形 | 引用该辅助图形的测量项删除或置为失效 |
| 未被引用的辅助图形 | 仅删除该辅助图形 |

## 12. 最终测量项归类

| 测量项 | 最终类别 | 依赖来源           | 创建方式        | 工具面板处理      |
|---|------|----------------|-------------|-------------|
| T1 Tilt | A 类  | T1 四角点         | 系统自动生成或展示   | 不开放         |
| CA | A 类  | CL、CR          | 系统自动生成或展示   | 不开放         |
