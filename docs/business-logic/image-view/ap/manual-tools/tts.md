# 胸廓躯干偏移（TTS）

## 适用范围

- 仅适用于正位 X 光片的手动标注。
- 工具名称为 `TTS`，含义为胸廓躯干偏移（Thoracic Trunk Shift）。
- 每张影像最多保留一个 TTS 测量项。
- TTS 距离需要使用影像的标准距离换算为毫米；未设置标准距离时，沿用标注页现有的标准距离提示流程。

## 四点结构

TTS 测量项保存 4 个点：

| 点位索引 | 来源 | 作用 |
| --- | --- | --- |
| 0 | 用户手工确定 | 躯干参考水平线端点 1 |
| 1 | 用户手工确定 | 躯干参考水平线端点 2 |
| 2 | SL、SR 对应的骶骨参考线 | 骶骨参考线端点 1 |
| 3 | SL、SR 对应的骶骨参考线 | 骶骨参考线端点 2 |

点 `0-1` 形成医生可调整的躯干参考水平线。点 `2-3` 继承自 CSS 使用的 SL、SR 骶骨参考线，不属于 TTS 的自由手工点。

## 创建规则

1. SL、SR 已确定时，系统继承骶骨参考线的两个点，医生只需要在影像上再点击两次。
2. 第一次点击确定躯干水平线的第一个端点。
3. 第二次点击确定第二个端点的 X 坐标，其 Y 坐标强制使用第一个端点的 Y 坐标。
4. 两个手工点与两个继承点齐全后，系统创建 TTS，并自动退出当前测量工具。

因此，无论医生第二次点击的位置高低如何，最终躯干参考线始终保持水平。

## 计算规则

- 躯干参考位置为点 `0-1` 的中点 X 坐标。
- 骶骨参考位置为点 `2-3` 的中点 X 坐标。
- 像素偏移量为“躯干中点 X - 骶骨中点 X”。
- 系统使用标准距离将偏移量的绝对值换算为毫米，再保留原始左右方向符号。
- 躯干中点位于骶骨中点左侧时结果为负，位于右侧时结果为正。

TTS 数值只由两条参考线的水平位置决定，单纯上下移动躯干水平线不会改变测量值。

## 图像显示

- 显示点 `0-1` 组成的躯干参考水平线。
- 经过躯干水平线中点向下显示垂直引导线。
- 显示点 `2-3` 组成的骶骨参考线，并经过其中点向上显示垂直引导线。
- 在两条垂直引导线之间显示水平距离箭头。
- TTS 数值同时显示在影像和左上角测量结果列表中。

## 调整规则

- 拖动点 `0` 或点 `1`：只调整当前端点的 X 坐标，Y 坐标锁定为另一端点的 Y 坐标，水平线不能被拖成斜线。
- 拖动躯干水平线主体：两个端点只沿垂直方向同步移动；两个端点的 X 坐标和线段长度保持不变。
- 拖动躯干水平线主体时，继承的点 `2-3` 保持不动。
- 点和线段重叠时，端点命中优先于线段主体。
- 骶骨参考线主体不响应 TTS 的整体拖动；需要调整骶骨参考时，应移动 SL、SR 或对应的 CSS 测量点。
- 一次从鼠标按下到松开的线段拖动记为一次标注历史操作，可以整体撤回。

## 历史数据兼容

历史数据中可能存在通过上下椎体中心派生、带 `upperVertebra` 和 `lowerVertebra` 绑定的 TTS。这类测量项继续由关键点重算，不开放本规则中的手工水平线主体拖动。
