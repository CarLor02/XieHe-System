# 半骨盆宽度比（L/R）

## 适用范围

- 仅适用于正位 X 光片的手动标注。
- 工具名称为 `L/R`，工具提示为“半骨盆宽度比(L/R)”。
- 每张影像最多保留一个 L/R 测量项。

## 解剖线定义

用户在影像上点击四个解剖点，每个点生成一条经过该点的垂直纵轴。点击顺序不影响最终含义；系统始终按纵轴在屏幕上的水平位置从左到右重新确定序号。

| 从左到右序号 | 名称与标签     |
| ------------ | -------------- |
| 1            | 左外缘（ASIS） |
| 2            | 左内缘（SI）   |
| 3            | 右内缘（SI）   |
| 4            | 右外缘（ASIS） |

## 数据结构

L/R 测量项保存 12 个点，将医生确定的解剖点与纵线交互端点分开：

- 点 `0-3`：医生依次点击的四个解剖锚点，保留原始点击分组，不按水平位置改写数组顺序。
- 点 `4-5`：锚点 0 对应纵线的上、下端点。
- 点 `6-7`：锚点 1 对应纵线的上、下端点。
- 点 `8-9`：锚点 2 对应纵线的上、下端点。
- 点 `10-11`：锚点 3 对应纵线的上、下端点。

排序、显示名称和计算均以锚点的 X 坐标为准。X 坐标相同时，以原始点击分组序号作为稳定排序条件。

## 计算规则

完成四次点击后，系统立即按屏幕从左到右取得四条纵线，并计算：

- `dL`：第 1、2 条纵线之间的水平距离。
- `dR`：第 3、4 条纵线之间的水平距离。
- `L/R = dL / dR`，结果无单位。

`dL` 和 `dR` 使用影像当前的标准距离标定换算为毫米，并显示在影像中；`L/R` 同时显示在影像中和左上角测量结果列表中。未设置标准距离时，沿用现有测量工具的默认距离换算规则。当 `dR` 为 0 时，L/R 显示为 `--`，不得产生 `Infinity` 或 `NaN`。

## 操作流程

1. 点击工具栏中的“半骨盆宽度比（L/R）”。
2. 在影像上连续点击四个解剖点。操作时不强制从左到右，计算时统一按水平位置排序。
3. 每次点击立即预览经过该点的垂直纵轴；第四次点击后生成完整测量，并显示 `dL`、`dR` 和 `L/R`。
4. 医生可拖动任意解剖锚点、纵线或纵线端点进行微调，数值实时更新。

## 调整规则

- 拖动解剖锚点：锚点和对应纵线的两个端点一起移动，纵线始终经过锚点。
- 拖动纵线主体：只允许水平移动，同时更新对应锚点及两个端点的 X 坐标。
- 拖动纵线端点：只改变端点的 Y 坐标，用于拉长或缩短纵线；端点 X 坐标锁定为锚点 X 坐标，且纵线范围始终包含锚点。
- 纵线调整后如果发生左右次序交换，系统立即按新的水平位置更新四条线的名称、`dL`、`dR` 和 `L/R`。
